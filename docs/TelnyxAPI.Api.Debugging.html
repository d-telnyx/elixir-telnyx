    list_call_events(connection, opts \\ [])

    List call events Filters call events by given filter parameters. Events are ordered by event_timestamp. If filter for call_leg_id or call_session_id is not present, it only filters events from the last 24 hours. Note: Only one filter[event_timestamp] can be passed.
